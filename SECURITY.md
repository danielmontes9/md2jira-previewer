# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| latest  | :white_check_mark: |

## Reporting a Vulnerability

**Do not open a public GitHub issue for security vulnerabilities.**

Please use [GitHub's Private Vulnerability Reporting](https://github.com/danielmontes9/md2jira/security/advisories/new) to report security issues.

When reporting, include:

- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

You will receive a response within 72 hours. We will coordinate a fix and disclosure timeline with you.
